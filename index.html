GitAchieve

A web application to promote positive Git behavior through friendly competition

Live Link: www.gitachive.com

Repo Link: github.com/msmith9393/GitAchieve

I worked on a team of four engineers to build a single page web application using React and Redux. We choose these tools, because it allowed us to centralize our state and use it in any of our components. This led to very module and easy to digest components even as our application grew to have more features. I worked across the full stack, designing a RESTful API in Node/Express that served custom data from GitHub. We leveraged GitHub’s API extensively and wrote Postgres SQL queries to store relational data from GitHub for our users in the database.

Evently

Event planning app for organizing events in real time

Repo Link: github.com/msmith9393/evently

Evently is an application that allows users to both create events and attend events created by other users. I worked with a team of four engineers to create Evently. We chose to use React for its fast re-rendering and modularity. We integrated the Google Maps API for displaying event locations as well as being able to add locations to the itinerary by clicking on the map. We stored all of our information about users and events in a MongoDB database, which made handling our data very easy in JSON format. Lastly, we used Socket.io to implement real time chat.

Technologies

  • React
  • React Router
  • Socket.io
  • Webpack

  • Node/Express
  • MongoDB
  • Google Maps API
